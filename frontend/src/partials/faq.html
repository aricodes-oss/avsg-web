FAQ

What is this?

This is a web-based tool to encrypt and decrypt your Axiom Verge save file.

When would I use this?

This tool is helpful for detecting and repairing corrupted save files, editing your progress, or just for curiosity.

Only save files from the Steam version are encrypted. The Epic games edition stores save files in plaintext.

Which save file is which?

There are a total of 5 save files that Axiom Verge can create. These files are all named SaveX.sav, where X is a number between 0 and 4 inclusive.

Save files 0, 1, and 2 are your story mode save files.

Save file 3 is your speedrun mode save, and 4 is your randomizer save.

Where is my save file located?

Unless you specified differently in Steam's options, your save files can be found in C:/Program Files(x86)/Steam/userdata/[Steam Account ID]/332200/remote.

~/.steam/steam/userdata/[Steam Account ID]/332200

~/Library/Application Support/[DeveloperGameName (which will probably be AxiomVerge?)]/SaveGames
